ppt2ncl
=======
ppt2ncl is a simple tool to convert PPT files to Nested Context Language (NCL)
documents.


### Dependencies
ppt2ncl depends on <a href="http://java.com">Java JRE 1.7</a> (or greater) and
on <a href="https://poi.apache.org/">POI 3.5</a> (or greater).
